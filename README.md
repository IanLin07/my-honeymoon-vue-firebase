# my-honeymoon-vue-firebase

蜜月行程 Web App（Vue 3 + Vite + Firebase），部署於 GitHub Pages。

---

## 技術架構

| 類別 | 使用技術 |
|---|---|
| 前端 | Vue 3 |
| 建構工具 | Vite |
| 身份驗證 | Firebase Authentication（Google 登入） |
| 資料庫 | Cloud Firestore |
| 部署 | GitHub Pages（gh-pages） |
| 套件管理 | npm |
